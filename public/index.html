Tentang Dashboard Ini

Dashboard ini merupakan tampilan publik sederhana dari sistem GridSight‚Ñ¢, dirancang untuk mendukung transformasi digital utilitas distribusi tenaga listrik. Versi internal menampilkan peta teknis (JTM, JTR, SR, titik pelanggan, dan aset pendukung), analitik spasial, serta integrasi ke database spasial terkelola.

Tujuan halaman publik: memberikan gambaran KPI distribusi tanpa menampilkan geometri peta, sehingga aman dibagikan untuk portofolio & showcase.

Potensi WebGIS Berbasis Cloud PLN

  • Monitoring aset distribusi secara near real-time & analisis pertumbuhan pelanggan.
  • Perencanaan pembangunan jaringan berbasis spasial (prioritas JTM/JTR/SR).
  • Integrasi pekerjaan lapangan (work orders) & pelacakan progres pada peta digital.
  • Pemanfaatan data IoT/SCADA/AMI dalam konteks geospasial untuk insight operasional.
  • Arsitektur cloud + database spasial memastikan kolaborasi lintas unit secara aman.
